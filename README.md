# WIDGETS

## Descripcion del Proyecto

Desarrolle una aplicación con el objetivo de poder mostrar el uso e implementación de diferentes widgets con sus respectivos comportamientos utilizando diferentes métodos y propiedades, con la opción de cambiar el tema de la aplicación, botones flotantes, el uso de GoRouter para la navegación entre pantallas con las implementaciones nativas de Flutter, tarjetas, Snackbars y cuadros de diálogos, implementación de diferentes Scrolls infinitos, tarjetas, entre otros.
***
## Tecnologias
Tecnologias que se utilizaron en el proyecto:
- Flutter 3.16.6
- Dart 3.2.3
- Xcode 15.2
- iOS 17.0
***
**Dependencies**
- animate_do: ^3.1.2
- flutter_riverpod: ^2.4.6
- go_router: ^12.0.1

**Environment**
- SDK: >= 3.1.5 <4.0.0

***
## Funcionalidades
Encontramos una aplicacion en donde implementamos el desarrollo de diferentes widgets: un contador, botones, tarjetas, indicadores de progreso, snackbars y cuadros de dialogos, contenedor animado, ui control, una introduccion de app (slide), scroll infinito, cambio de tema de la app y el comportamiento respectivo de cada uno y sus diferentes estados. Se implemento un menu con su respectiva navegacion hacia cada uno de los diferentes widgets utilizando el metodo GoRouter.
